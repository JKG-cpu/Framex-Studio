# Framex Studio
This is a GUI Game Engine for my already built Framex game engine

# Modules

- PySide6
- framex-ce

# How to use?

Clone the repo
```bash
git clone https://github.com/JKG-cpu/Framex-Studio
cd Framex-Studio
```

Run with uv
```bash
uv sync
uv run main.py
```

OR Install packages with python
```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
pip install -r requirements.txt
python main.py
```
